# Advanced Data Mining on IMDb Dataset

## Overview
This project was developed for the *Data Mining: Advanced Topics and Applications* course as part of my Master's degree in Data Science and Business Informatics at the University of Pisa.

The project focuses on the analysis of IMDb datasets, including both structured data and time series data.

It covers advanced data mining topics such as outlier detection, imbalanced learning, classification, regression, explainability, and time series analysis, with the goal of exploring patterns and supporting predictive modeling.
